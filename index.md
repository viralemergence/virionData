# virionData

The goal of virionData is to provide a consistent and ergonomic way to
access versions of the [VIRION data set stored in
Zenodo](https://doi.org/10.5281/zenodo.15643003).

This package allows users to select specific versions of the data set to
work with, provides easy access to metadata, and allows users to export
metadata.

The pipeline for creating virion can be found here:
<https://github.com/viralemergence/virion>

Python users can use
[py-virion-data](https://pypi.org/project/py-virion-data/).

## Installation

You can install the development version of virionData from
[GitHub](https://github.com/) with:

``` r

# install.packages("remotes")
remotes::install_github("viralemergence/virionData")
```
